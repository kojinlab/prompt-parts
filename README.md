# Prompt Parts

AIへの定番指示をパーツ化して、選んで組み立てて、すぐコピーできるPWAです。

## スマホで確認できる公開方法

このアプリはローカルサーバーではなく、GitHub Pagesで公開して確認する前提です。

1. このフォルダをGitHubリポジトリにする
2. `main` ブランチにpushする
3. GitHubのリポジトリ設定で Pages を `GitHub Actions` にする
4. Actionsの `Deploy to GitHub Pages` が完了したら、表示されたURLをスマホで開く

`localhost` や `127.0.0.1` はMacの中だけのURLなので、スマホ遠隔チェックには使いません。

ビルド工程はありません。静的ファイルだけで動きます。

## MVP機能

- プロンプトパーツのカテゴリ表示
- 検索
- クリックで組み立て
- 並び替え
- 追加メモ
- ワンクリックコピー
- パーツ追加
- ローカル保存
- JSON書き出し/読み込み
- GitHub README / raw Markdown URLから候補取り込み
- PWA対応

## SNSでの見せ方

「プロンプトを保存するアプリ」ではなく、以下のように見せるのが伝わりやすいです。

> 毎回AIに打ってる「まず質問して」「いきなり実行しないで」「否定的にも見て」を、チェックして組み立てるだけの無料PWAを作りました。
